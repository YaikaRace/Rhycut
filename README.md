# Rhycut
Rhycut is a rhythm game inspired by the mechanics of the popular mobile game Fruit Ninja.

## About
In this game, you must keep up with the beat of the music by slicing the objects that are thrown at you from below and avoid the bombs at all costs!

It comes with a built-in BeatMap Editor so you can create your own maps and share them with your friends!

## Features
This beta includes the following features:

- 1 Playable map
- Built-in BeatMap Editor
- BeatMap Modifiers
- Audio Calibration Settings
